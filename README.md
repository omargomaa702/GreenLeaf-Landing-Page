# 🌿 GreenLeaf Landing Page

A modern and responsive landing page for a plant store.
It showcases premium indoor plants with a clean UI and smooth user experience.

---

## 🚀 Features

* Responsive design
* Product showcase section
* Add to cart UI
* Clean and minimal layout
* Smooth hover effects
